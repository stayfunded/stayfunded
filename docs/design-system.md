# StayFunded — Design System (Canonical)

STATUS: CANONICAL  
Last updated: 2025-01-XX  
Do not relitigate visual decisions without explicitly updating this document.

---

## 1. Design Philosophy

StayFunded should feel:

- **Serious, not flashy**
- **Insider, not marketing**
- **Clear, not clever**
- **Calm under pressure**

This is not a trading app.  
It is a **decision support system** for constrained environments.

The UI should feel like:
- A cockpit
- A briefing room
- A classified manual
- A flight plan

Never:
- “Growth SaaS”
- Neon crypto
- Loud gradients
- Gamification

---

## 2. Color System

### Core Backgrounds
- **Primary dark**: `#050815`
- **Secondary dark**: `#070A18`
- **Tertiary dark**: `#0B1022`
- **Primary light**: `#FFFFFF`
- **Light surface**: `#F8FAFC`
- **Muted surface**: `#F1F5F9`

### Accent Colors
- **Primary accent (indigo)**: `#6366F1`
- **Secondary accent (amber)**: `#FBBF24`
- **Danger**: `#DC2626`
- **Warning**: `#F59E0B`
- **Success**: `#16A34A`

### Rules
- Dark gradients are allowed **only** in hero sections and callouts.
- Most interior pages are light background.
- Accent colors are used sparingly for emphasis, not decoration.

---

## 3. Typography

### Font Stack
- **Primary**: Inter (system fallback acceptable)
- **No decorative fonts**

### Type Scale
- H1: 40–48px, semibold
- H2: 30–36px, semibold
- H3: 22–26px, semibold
- Body: 15–17px, regular
- Small: 12–13px

### Copy Rules
- Short sentences.
- One idea per paragraph.
- Prefer blunt clarity over clever phrasing.
- Avoid jargon unless it’s unavoidable and explained.

---

## 4. Layout System

### Max Widths
- Content max width: **1200px**
- Reading content: **800–900px**

### Spacing
- Section spacing: 96px desktop / 64px mobile
- Card padding: 20–24px
- Grid gaps: 16–24px

### Grid
- Desktop: 12-column grid
- Mobile: stacked

---

## 5. Components

### Buttons
- Primary: solid, rounded-xl
- Secondary: outlined
- No ghost buttons for CTAs
- Hover = subtle brightness shift only

### Cards
- Rounded-xl or 2xl
- Light border
- Soft shadow
- Never floating neon shadows

### Tags / Pills
- Small
- Muted
- Used to label concepts, not decorate

---

## 6. Diagrams & Visuals

### Lifecycle / Flight Plan Diagrams
- Always framed
- Always labeled
- Wide and short aspect ratios
- Treated as **reference artifacts**, not decoration

### Images
- Prefer diagrams over stock photos
- If photos are used: grayscale or muted

---

## 7. Tone Guardrails (UI)

Allowed:
- “This rule exists because…”
- “Most traders fail here.”
- “This is where accounts die.”

Avoid:
- “Optimize”
- “Leverage alpha”
- “Edge”
- “Crush”
- “Hack”
